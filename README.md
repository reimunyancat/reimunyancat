## Things I've built

- **[Ephemeris](https://github.com/reimunyancat/Ephemeris)** — a 3D solar-system simulator that moves planets with real N-body gravity (octree-optimized), seeded from NASA/JPL data.
- **[AudiLex](https://github.com/reimunyancat/AudiLex)** — transcribes & translates audio and YouTube with Whisper + LLMs (Django + FastAPI, WebSockets).
- **[CrashZero](https://github.com/reimunyancat/CrashZero)** — predicts road-accident risk and simulates countermeasure budgets on Korean open data; the ML (logistic regression + gradient boosting) is written from scratch in NumPy.
- **[enigma](https://github.com/reimunyancat/enigma)** — an interactive 3D Enigma cipher machine in the browser; one C++ engine shared by a CLI and WebAssembly (Svelte + Three.js).
- **[Flit](https://github.com/reimunyancat/Flit)** — a frictionless cross-device drop inbox in pure Rust (axum): send text, links, or files between devices with no accounts and no cloud.
